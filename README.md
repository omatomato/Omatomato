##  Descrição
Repositório central do projeto **Dead Trace**. Este é um laboratório de pesquisa focado em soberania digital, criptografia e análise de segurança. O projeto serve como dashboard institucional e base para disseminação de relatórios técnicos.

##  Tech Stack
Este projeto é construído seguindo uma filosofia de **minimalismo extremo** e **segurança nativa**:
- **100% HTML5 & CSS3**: Zero JavaScript no core para garantir máxima performance e redução de vetores de ataque (XSS-free by design).
- **No Frameworks**: Sem Bootstrap, Tailwind ou bibliotecas externas. Código puro, estruturado para baixo consumo de recursos.
- **OpSec Focused**: Implementação de cabeçalhos de segurança (CSP), política `security.txt` e chaves PGP.

## Segurança (Security Policy)
A integridade é prioridade absoluta. Você pode verificar a autenticidade dos meus arquivos e comunicações através dos seguintes protocolos:

- **PGP Key:** Disponível em [`pgp.txt`](./pgp.txt) ou via `/.well-known/security.txt`.
- **Fingerprint:** `2B43 C0CE 100D 8B6F 76FB C2BB 765C 151A 678A 5A34`
- **Manifest:** Todos os relatórios são assinados digitalmente e possuem hash SHA-256 verificado.

## Estrutura do Projeto
- `index.html`: Dashboard principal e interface de navegação.
- `service-detail.html`: Detalhamento técnico de serviços e pesquisa.
- `.well-known/`: Implementação de padrões globais de segurança (RFC 9116).
- `assets/`: Identidade visual (Charcoal Black & Acid Green).
